jQuery Right-click Event Reference

A jQuery plugin that allows you to bind to a right click or multiple-click event.

This page provides a documentation reference for working with the plugin v1.0.0. See a demonstration of the right-click plugin and download the code from there.

  • Usage
  • Multiple-click

Usage

Just bind to the rightclick event as you would normally.

$(selector).bind('rightclick', function(event) { ... });

You can remove an event handler with the unbind function. You can also use on and off for the same purposes. Include namespaces for your events if desired.

The event.type available to the callback function is 'rightclick'.

Your callback function should return false to suppress the default behaviour, which opens a context menu.

You can disable the right-click functionality on individual elements by adding them to the $.event.rightclickDisabled collection.

$.event.rightclickDisabled = $.event.rightclickDisabled.add('#rightClick');

Re-enable them by removing them from this set.

$.event.rightclickDisabled = $.event.rightclickDisabled.filter(':not(#rightClick)');

Multiple-click

Bind to the rightmulticlick event and configure it as necessary.

$(selector).bind('rightmulticlick', function(event) { ... });
$(selector).bind('rightmulticlick', {clickCount: 3, clickSpeed: 300},
	function(event) { ... });
NameTypeDefaultComments
clickCountnumber2 The number of consecutive clicks before the event fires.
clickSpeednumber500 The maximum time for all clicks to count as a multiple click.

Adding jQuery Selectors

jQuery lets you extend much of its functionality, including adding new selectors.

jQuery has a contains selector, but that matches anywhere within the text content. We can add a selector (content) that checks the complete content for an exact match only, and another (matches) that matches content against a regular expression.

We add a list selector that matches both ordered (ol) and unordered (ul) lists, and a middle selector that matches items other than the first or last in a set.

We could also replace an existing selector to enhance its abilities, such as allowing the eq selector to work from the end of the list by specifying a negative number.

The example below shows how they work, being applied within the main list.
